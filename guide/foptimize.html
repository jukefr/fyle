# foptimize

Specialized optimisation and compression pipelines

# 🔧 Formats

  • gif
  • png
  • jpg
  • svg

# 🐳 gif

  • ✅ giflossy
$ docker run -v $(pwd):/d/ foptimize/gif input.gif
$ docker run -v $(pwd):/d/ foptimize/gif "https://...gif"

name (default)

  1. file
  2. quality (80)
  3. colors (128)

# 🐳 png

  • ☑️ pngcrush
